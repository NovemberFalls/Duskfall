# The Green Guard Path Structures and Authority

*"Each path is a different dance to the same eternal song."*  
— High Commander Aelindra Starweaver

## Path Overview

### Spellblade Path
*"The dance of blade and magic as one."*

**Core Focus:** Combat integration of martial prowess and magical ability

**Circle Ranks:**
- Elder Circle: Blade-Master
- Sentinel Circle: Blade-Singer
- Guardian Circle: Spellblade
- Initiate Circle: Blade Dancer
- Base Circle: Path Seeker

**Core Powers:**
- Weapon enchantment
- Combat magic
- Battlefield mobility
- Tactical leadership

### Grove Guardian Path
*"The shield between nature and chaos."*

**Core Focus:** Defense of sacred sites and natural barriers

**Circle Ranks:**
- Elder Circle: Grove-Keeper
- Sentinel Circle: Root-Keeper
- Guardian Circle: Grove Guardian
- Initiate Circle: Grove Warden
- Base Circle: Path Seeker

**Core Powers:**
- Natural barrier creation
- Environmental manipulation
- Sacred site protection
- Defensive magic

### Starweave Path
*"The weavers of starlight and shadow."*

**Core Focus:** Pure magical combat and ritual magic

**Circle Ranks:**
- Elder Circle: Star-Sage
- Sentinel Circle: Star-Weaver
- Guardian Circle: Starweaver
- Initiate Circle: Star Apprentice
- Base Circle: Path Seeker

**Core Powers:**
- Battle magic
- Ritual casting
- Magical detection
- Strategic support

## Authority Structure

### Circle Authority
1. **Elder Circle**
   - Supreme authority in path specialty
   - Direct path development
   - Oversee major operations
   - Traditional knowledge preservation

2. **Sentinel Circle**
   - Regional command
   - Major site oversight
   - Advanced training
   - Tactical deployment

3. **Guardian Circle**
   - Unit leadership
   - Direct combat roles
   - Site protection
   - Training supervision

4. **Initiate Circle**
   - Basic leadership
   - Regular duties
   - Training assistance
   - Site maintenance

5. **Base Circle**
   - Core duties
   - Basic training
   - Support roles
   - Path discovery

### Cross-Path Integration

**Combat Operations:**
1. Spellblade Path leads direct engagement
2. Grove Guardian Path maintains defenses
3. Starweave Path provides magical support

**Defensive Operations:**
1. Grove Guardian Path leads protection
2. Spellblade Path provides combat support
3. Starweave Path maintains magical barriers

**Magical Operations:**
1. Starweave Path leads magical work
2. Grove Guardian Path maintains stability
3. Spellblade Path provides security

1. **Initial Assessment**

2. **Path Trials**
   - Specific skill tests

3. **Final Assignment**

**Guardian Circle**
